# Minecraft Autoclicker

This is a Python-based autoclicker for Minecraft with a GUI. It can be configured to specify the left-click interval and toggle whether to hold right-click.

This started out as a simple fork of [sohumcs/auto-clicker](https://github.com/sohumcs/auto-clicker) that I modified specifically for Minecraft. I kept developing it and adding features to where it now barely resembles the original.
